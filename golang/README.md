I wanted to do async programming with GoLang and found [Goroutines](https://tour.golang.org/concurrency/1). At first I wanted to _create a server_ on go, but didn't find appropriate lessons. But found an article on Medium and do this project with the following code.

## Features

With this program you can **check** web-urls asynchronously.

## Installation

1. Clone this repository:

```bash
git clone https://github.com/mezgoodle/async_server.git
```

2. Install golang from [here](https://golang.org/dl/)

## Fast usage

1. Move to _golang_ folder and execute the program:

```bash
cd golang
go run main.go
```
